# diabetes-prediction
A Diabetes Prediction WebApp using Machine Learning and Python.

* Modules Used:
  - pandas
  - sklearn.metrics
  - sklearn.model_selection
  - sklearn.ensemble
  - PIL
  - streamlit

* Data used:
  - Diabetes.csv

* To run the project on terminal:
  - `source ~/.bash_profile`
  - `streamlit run {include path of directory here}`
